# Blazor Books Sample App（Part3）

このリポジトリは、技術書 **「Blazor実践入門｜C#で作るWebアプリケーション」**  の **Part3（Blazor実践編）** で作成する、書籍管理アプリの完成形のソースコードです。

## このリポジトリについて

- 本書 Part3 の内容に対応した **完成形コード** です
- 学習中に詰まったときの **確認用・参考用** として公開しています
- **このリポジトリだけでの理解は想定していません**
- 実装の詳細な解説は、書籍本文を参照してください

## 対象範囲

- Part3：Blazor実践編  
  （CRUDアプリの実装、サービス層、一覧改善、リファクタリング）

※ Part1（C#基礎編）、Part2（Blazor基礎編）のコードは含まれていません。

## 動作環境

- .NET 10（または .NET 8 以降）
- ASP.NET Core Blazor（Interactive Server）
- SQLite
- Visual Studio 2022 以降（推奨）

## 起動方法（簡易）

1. このリポジトリをクローンします
2. Visual Studioでソリューションを開きます
3. 実行（F5）すると、SQLiteのデータベースが自動的に作成されます

※ 詳細な手順や設定の説明は書籍内で解説しています。
